# Variant image folders

Use lowercase folder names:

- small
- medium
- large
- collector
- super-collector

Each folder uses JPG files:

- front.jpg
- back.jpg
- top.jpg
- bottom.jpg
- certificate.jpg
- xray.jpg

Example:

public/images/products/13-mukhi-nepal-rudraksha/small/front.jpg

The original product-level `.jpeg` images remain as fallbacks if a size-specific `.jpg` file is missing.
